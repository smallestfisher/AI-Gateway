## 11. 前端工程约定

- 目录：`apps/web/`（Next.js）；`app/(admin)/<module>/page.tsx`。
- 数据：TanStack Query 拉 `/api/admin/*`；写操作后 invalidate 对应 query。
- 表格：`@tanstack/react-table` + shadcn；表单：`react-hook-form` + `zod`。
- 鉴权：登录页 → JWT 存 httpOnly cookie；中间件守卫 `/admin`。
- 实时：Dashboard/健康页用 SWR 轮询（5s）；日志可用 SSE 推送 tail。
